# f-scroll
Add animate into block when appeared using appear: https://github.com/morr/jquery.appear/
## Import
Css 
```html
<link rel="stylesheet" href="css/animate.css" />
<link rel="stylesheet" href="css/style.css" />
```
Js
```html
<script src="https://raw.githubusercontent.com/morr/jquery.appear/master/jquery.appear.js"></script>
<script src="js/f-scroll.js"></script>
```
Then add class <code>f-scroll--fadeInDown</code> into your markup:
</br></br>See animate class name at: http://daneden.github.io/animate.css/
